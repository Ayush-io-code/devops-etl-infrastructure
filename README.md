
# DevOps Home Lab — Ayush Pareek

End-to-end DevOps project built on CentOS VM with 4GB RAM.

Replicates enterprise ETL infrastructure using open-source tooling.

## Architecture
Airflow (ETL) → Docker Compose
Ansible        → Automated provisioning
Prometheus     → Metrics collection
Grafana        → Dashboards & alerting
GitHub Actions → CI/CD pipeline

## How This Maps to My Work Experience
In my current role I manage Ab Initio ETL infrastructure on physical
servers. This lab replicates those patterns using modern open-source
tools — Airflow mirrors Ab Initio scheduling, Ansible mirrors manual
server provisioning, and Prometheus mirrors AppDynamics monitoring.
